Road-Check Privacy Policy

Last Updated: June 2024

1. Information We Access

Road-Check retrieves the following public data from Facebook groups through Facebook's API:

  • Group names and descriptions
  • Post counts and metadata
  • Publicly visible post content (text only)

We do not collect:

  • Private group data
  • User profiles or personal information
  • Images or videos

2. Data Storage

Road-Check does not store any group data. All information is:

  • Processed in real-time via Facebook's API
  • Displayed temporarily in the app interface
  • Never saved to databases, servers, or log files

3. Purpose of Data Access

The accessed public group data is used solely for:

  • Group analytics and statistics
  • Content monitoring (if applicable)
  • Research purposes

4. Data Deletion Instructions

Does Road-Check Store My Data?

No. Since Road-Check only accesses public data in real-time without storage:

  • No user data is retained
  • No deletion requests are necessary
  • Facebook maintains all original group data

5. Changes to This Policy

If Road-Check's data practices change in the future, we will:

  • Update this policy
  • Notify users through the app
  • Clearly document any new data storage
